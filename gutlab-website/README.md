# Gut Lab by Xaels Website

This is a deployable Vite + React website for Gut Lab by Xaels.

## 1) Add your real images
Put these files into the `public` folder using these exact names:

- `public/logo.png`
- `public/healthy_spread.jpg`
- `public/coffee_spread.jpg`
- `public/cocoa_mix.jpg`

## 2) Run locally
```bash
npm install
npm run dev
```

## 3) Build
```bash
npm run build
```

## 4) Deploy on Vercel
- Create a GitHub repo
- Upload this project
- Import the repo into Vercel
- Click Deploy
